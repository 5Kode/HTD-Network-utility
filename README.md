HTD Network Utility is a small, basic network scanner and capturer. I originally built this for myself, but came up with the idea to share it here.
HTD has a few features, and even though not overly useful, still handy to have around.

Here are the features:

Processes I/O, a table of your current running processes on your system, with PID, name, and IP, outgoing and incoming.
Devices Scanner, a module that simply ARP sweeps all of the active devices on your LAN and reports them back with an IP, MAC, and estimated hostname.
WLAM shows nearby WiFi networks and displays their security type, like WPA2 or WPA3, and the signal strength. You can also directly connect with my custom-built expanded menu.
Network I/O captures incoming and outgoing IP signals, and lists them in a table.
IP Tripwire. (Semi-working) The IP Tripwire tab looks at your Network I/O table, and if it sees a blacklisted IP, it will make a log in the designated area.

There is also a small export button in the top-right menu bar, which creates a file in the /logs folder, containing every line of data captured in that session.
Please run the code as administrator if the UAC prompt doesn't pop up, as it gives you access to over half of the app.
